Project Description

This project is a responsive user authentication system that allows users to register (signup) and login using their credentials. The system includes basic form validation to ensure correct user input and provides a user-friendly interface.

--> Key Features

User Signup and Login functionality

Basic validation (empty fields, password length, credential matching)

Responsive and clean UI design

Data storage using browser localStorage

Proper folder structure and clean coding practices

--> Technologies Used

HTML – Structure of the web pages

CSS – Styling and responsive layout

JavaScript – Logic, validation, and authentication handling

--> Working Flow

User registers using the signup page

User details are validated and stored

User logs in using registered credentials

System verifies credentials and allows access
